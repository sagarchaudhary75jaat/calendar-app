# Calendar App

A React-based calendar app that allows users to:

- Select a range of days
- Add and view notes for each day (stored in `localStorage`)
- Navigate between months
- Use a responsive UI on both desktop and mobile screens

---

## Features

- **Day Range Selection:** Easily select a start and end day.  
- **Notes Feature:** Save notes for any day; notes persist using `localStorage`.  
- **Month Navigation:** Switch between months to plan ahead.  
- **Responsive Design:** Works on desktops, tablets, and mobile devices.
